
1) Install dependencies > check dependencies.txt
2) Configure android files to ensure react-navigation works properly
3) do npm ls to verify that you have everything installed [should be similar enough]



Used to generate basic react app:
npx @react-native-community/cli@latest init drawzAllProject

- Deleted typescript config file [not necessary but did it to ensure it doesnt conflict]

React Navigation && React Native:
npm install @react-navigation/native
npm install @react-navigation/drawer
npm install react-native-reanimated
npm install react-native-screens
npm install react-native-gesture-handler
npm install react-native-worklets

Babel.config.js parser issue:
npm install react-native@latest metro-react-native-babel-preset@latest



Add this to your babel.config.js:
module.exports = {
  presets: ['@react-native/babel-preset'],
  plugins: [
    'react-native-worklets/plugin', 
  ],
};



Make sure this is added in MainActivity.kt at drawzAllProject\android\app\src\main\java\com\drawzallproject\MainActivity.kt

import android.os.Bundle;

^^This should be at the top of the file^^

  override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(null)
  }

^^Inside the class you find in MainActivity.kt, add this function! ^^^^
